## 项目使用方法：
    - 安装hugo，<https://github.com/gohugoio/hugo>
    - 下载<https://github.com/CaiJimmy/hugo-theme-stack>主题并解压到themes目录下

## 本地运行调试
```bash
# windows 
./run.bat

# linux
./run.sh
```

## 发布到githu.io
```bash

# windows
# 该脚本将生成静态文件到docs文件夹（需要在GitHub仓库中设置为docs）
./build.bat

# 推送到github

git add .

git commit -m 'your commit message'

git push

```
